# UI/UX Pro Max

This skill provides a standardized "Pro Max" visual language for modern web applications.

## Light Mode Color Scheme (Mealsxu Edition)

Use these HSL variables for a clean, fresh, and appetizing light mode.

```css
:root {
  --background: 0 0% 100%;
  --foreground: 20 14% 4%;
  --card: 0 0% 100%;
  --card-foreground: 20 14% 4%;
  --popover: 0 0% 100%;
  --popover-foreground: 20 14% 4%;
  --primary: 24 95% 53%; /* Energizing Orange */
  --primary-foreground: 60 9.1% 97.8%;
  --secondary: 60 4.8% 95.9%;
  --secondary-foreground: 24 9.8% 10%;
  --muted: 60 4.8% 95.9%;
  --muted-foreground: 25 5.3% 44.7%;
  --accent: 60 4.8% 95.9%;

  --destructive-foreground: 60 9.1% 97.8%;

```

## Typography & Spacing
- **Font:** Inter or System Sans-Serif.
- **Headings:** Semi-bold or Bold with tighter letter spacing (`tracking-tight`).
- **Cards:** Subtle borders (`border-border/50`) and soft shadows.
- **Interactive:** Gentle scaling on hover (`hover:scale-[1.02]`) and smooth transitions.
